# RejuvFR v1.0

Première publication publique. First public release.

## Compatibilité / Compatibility

Testé sur **Pokémon Rejuvenation 14.0 (Windows)**. Non compatible avec la 13.5.

Tested on **Pokémon Rejuvenation 14.0 (Windows)** only. Not compatible with 13.5.

## Installation

1. Télécharger `RejuvenationFR-1.0.zip` ci dessous.
2. L'extraire à la racine du dossier du jeu (où se trouve `Rejuvenation.exe`). Le dossier `patch` fusionne avec celui déjà présent.
3. Lancer `Rejuvenation.exe`, puis dans le menu principal choisir **Language** puis **Français**.

1. Download `RejuvenationFR-1.0.zip` below.
2. Extract it at the root of your game folder (where `Rejuvenation.exe` is). The `patch` folder merges with the existing one.
3. Launch `Rejuvenation.exe`, then in the main menu pick **Language** and **Français**.

## Coverage

Story, Pokédex, moves, items, abilities, UI, battles. Official French Pokémon terminology and Rejuvenation canon.

Le mod respecte le système de mods officiel décrit dans `Modding.txt`: aucun fichier de base du jeu n'est modifié. Fully compliant with the official mod system described in `Modding.txt`: no base game file is modified.
